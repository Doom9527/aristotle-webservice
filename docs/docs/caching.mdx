# 缓存机制

Aristotle 支持高效的缓存机制，提升大规模知识图谱的查询性能。

## 1. Caffeine 本地缓存

- 默认启用
- 支持按内存或条目数限制
- 通过 `spring.read-cache.enabled` 开关

```yaml
spring:
  read-cache:
    enabled: true
    num-subgraphs: 500
```

## 2. Redis 分布式缓存

- 可选启用
- 适合多实例/分布式部署
- 通过 `spring.redis.enabled` 开关

```yaml
spring:
  redis:
    enabled: true
    host: 127.0.0.1
    port: 6379
```

## 3. 缓存策略

- 支持 LRU、过期时间等策略
- 查询优先命中本地缓存，再查 Redis，最后查数据库
